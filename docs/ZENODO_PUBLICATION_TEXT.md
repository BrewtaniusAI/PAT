# Zenodo Publication Text

PAT is published as executable prior art and patent-free language infrastructure.

This release is part of a broader patent-free science and open collaboration framework. It is intended to function as public infrastructure, reproducible technical disclosure, and a foundation for community stewardship of African language AI tooling.

Included in this release:
- runnable PAT core
- public language profiles
- schema and sample corpus
- dataset building tools
- receipts and proof bundles
- release integrity checks
- community contribution files
